# AI Travel Planner (LLM-Powered)

A production-oriented **AI Travel Planner** that generates **short, actionable travel itineraries** using a Large Language Model (LLM).

The application converts structured user inputs (budget, days, group size, accommodation type, travel purpose, etc.) into a clear, budget-aware trip plan while minimizing API token usage and ensuring robust error handling.
